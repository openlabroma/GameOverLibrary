GameOverLibrary

Handful Processing Library for Rapid Game Developments

Download this project as a .zip file Download this project as a tar.gz file

This is a library for rapid prototyping of games and apps.

Old-school java with the fundamentals of Processing united in a simple yet powerful solution.

Ideal for game jams and hackatons, comes with a lot of examples!

Just unpack the zip file into your libraries/ folder inside your sketchbook. As usually ;-)
